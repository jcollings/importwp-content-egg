Requires Import WP: 2.5.0
Requires Import WP PRO: 2.5.0

**Version: 0.0.2**

## Description

Import WP Content Egg Offer Importer allows you to import offers from xml or csv files.

**This plugin requires Import WP PRO v2.5.0 or newer**
